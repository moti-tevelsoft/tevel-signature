איך להגדיר את החתימה

Gmail (אינטרנט)

  1. לחץ על "העתק ללוח" בראש העמוד.
  2. פתח את Gmail, לחץ על גלגל השיניים ובחר "הצג את כל ההגדרות".
  3. בלשונית "כללי", גלול עד לאזור "חתימה".
  4. צור חתימה חדשה (או בחר חתימה קיימת לעריכה).
  5. הדבק לתוך תיבת החתימה (Ctrl+V / Cmd+V).
  6. גלול לתחתית העמוד ולחץ "שמור שינויים".

Outlook (אינטרנט)

  1. לחץ על "העתק ללוח".
  2. בתוך Outlook, פתח הגדרות (גלגל שיניים) → "הצג את כל הגדרות Outlook".
  3. בחר דואר → כתוב והשב → חתימת דוא"ל.
  4. צור חתימה חדשה והדבק בתוך עורך החתימה.
  5. סמן את החתימה כברירת מחדל להודעות חדשות ולתשובות, ושמור.

Outlook (תוכנת שולחן עבודה ל-Windows)

  1. לחץ על "העתק ללוח".
  2. ב-Outlook לך לקובץ → אפשרויות → דואר → חתימות.
  3. לחץ "חדש", תן שם לחתימה והדבק לתוך עורך החתימה.
  4. בחר את החתימה כברירת מחדל להודעות חדשות ולהשבות, ולחץ אישור.

טיפ: אם ההדבקה משטיחה את העיצוב, בדוק שאתה מדביק בעורך עשיר (Rich text / HTML) ולא במצב טקסט בלבד.
