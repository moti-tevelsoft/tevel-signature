איך להגדיר את החתימה

Gmail (אינטרנט)

  1. לחץ על "העתק ללוח".
  2. פתח את Gmail, לחץ על גלגל השיניים ובחר "הצג את כל ההגדרות", או לחץ כאן.
  3. בלשונית "כללי", גלול עד לאזור "חתימה".
  4. צור חתימה חדשה (או בחר חתימה קיימת לעריכה).
  5. הדבק לתוך תיבת החתימה (Ctrl+V / Cmd+V).
  6. תחת "ברירות מחדל לחתימה", הגדר את החתימה רק עבור "להודעות חדשות בשימוש". בשדה "במצב מענה/העברה בשימוש" השאר "ללא חתימה".
  7. גלול לתחתית העמוד ולחץ "שמור שינויים".

Outlook (אינטרנט)

  1. לחץ על "העתק ללוח".
  2. בתוך Outlook, פתח הגדרות (גלגל שיניים) ← "הצג את כל הגדרות Outlook".
  3. בחר דואר ← כתוב והשב ← חתימת דוא"ל.
  4. צור חתימה חדשה והדבק לתוך עורך החתימה.
  5. תחת "בחר חתימות ברירת מחדל", הגדר את החתימה רק עבור "להודעות חדשות". בשדה "להשבות/העברות" השאר "ללא חתימה". שמור.

Outlook (תוכנת שולחן עבודה ל-Windows)

  1. לחץ על "העתק ללוח".
  2. ב-Outlook לך לקובץ ← אפשרויות ← דואר ← חתימות.
  3. לחץ "חדש", תן שם לחתימה והדבק לתוך עורך החתימה.
  4. תחת "בחר חתימת ברירת מחדל", בחר את החתימה רק בשדה "הודעות חדשות". בשדה "תשובות/העברה" השאר "ללא". לחץ אישור.

טיפ: אם ההדבקה משטיחה את העיצוב, בדוק שאתה מדביק בעורך עשיר (Rich text / HTML) ולא במצב טקסט בלבד.
